## Protocolo Anti-Distracção

Quando aparece uma ideia nova, pergunta-te:

```
1. "Isto serve o MVP da FYVR AGORA?"
   ├── SIM → Integra no backlog existente (fala com o PM)
   └── NÃO → Escreve no Parking Lot ↑ e volta ao trabalho

2. "Isto é urgente ou só parece urgente?"
   ├── URGENTE (bug, crash, perda de dados) → Age agora
   └── PARECE URGENTE (nova feature, nova ideia) → Parking Lot

3. "Daqui a 3 meses, isto ainda vai ser importante?"
   ├── SIM → Está seguro no Parking Lot para o quarterly
   └── NÃO → Então não merece o teu tempo agora
```
