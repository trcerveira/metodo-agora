## PARTE 1: A Equacao de Valor do Hormozi

### 1.1 A Formula

```
                    RESULTADO DESEJADO x PROBABILIDADE DE SUCESSO
VALOR PERCEBIDO = ──────────────────────────────────────────────────
                    TEMPO ATE RESULTADO x ESFORCO/SACRIFICIO
```

Para aumentar o valor percebido:
- **AUMENTAR** o resultado desejado (sonho maior)
- **AUMENTAR** a probabilidade de sucesso (mais provas, mais garantias)
- **DIMINUIR** o tempo ate resultado (mais rapido)
- **DIMINUIR** o esforco/sacrificio (mais facil)

```
  (minus affiliate payouts: ~$25.000)

LTV MEDIO = $302.80 por trial signup
```
