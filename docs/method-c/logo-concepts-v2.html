Concept 1
Geometric C — Diamond Cut
O "C" tem cortes geométricos internos — mesmo approach do "O" na referência. O C funciona como ícone standalone. Angular, técnico, moderno.

Concept 1

  • Mesmo approach da referência — letra com formas geométricas internas
  • Diamond/cristal dentro do C — sugere precisão, transformação
  • O C funciona isolado numa camisola como ícone
  • Angular e técnico — estética desportiva
★ Recomendado
Progress C — 3 Levels Built In
O "C" é construído por 3 barras horizontais que formam a letra — cada barra = um nível (Corrige, Constrói, Compete). Progressivo, limpo, conta a história da marca.
Full Wordmark
Icon — Clean
Icon — Reversed
Icon — P&B
3 Cores = 3 Níveis

Porquê este é o recomendado

  • Conta a história: 3 barras = Corrige (base), Constrói (meio), Compete (topo)
  • Lê-se como C: mesmo simplificado, lê-se como a letra C
  • Standalone: funciona numa camisola, boné, app icon
  • Mesma estética da referência — bold, geométrico, desportivo
  • Versão multicolor: pode usar 3 cores para os 3 níveis quando quiser
  • Escalável: funciona de 16px a billboard
