METHOD C

Draplin Design Co. Methodology

Option A — Safe
The Clean Wordmark
Tipografia bold e directa. O "C" ganha peso extra como elemento de destaque. Sem ícone — a força está na tipografia. Profissional, escalável, limpo.

Notas Draplin

  • Wordmark puro — sem distracções
  • Linha inferior ancora o logo e cria estrutura
  • "C" ligeiramente maior — hierarquia visual
  • Funciona em qualquer tamanho como texto
  • Seguro, profissional, impossível de errar
★ Recomendado
The 3-Level C
Um "C" bold construído a partir de 3 segmentos — cada um representa um nível do método (Corrige, Constrói, Compete). A forma do C mantém-se, mas conta a história da marca. O ícone funciona sozinho ou com o wordmark.

Icon Standalone

  • O "C" funciona sozinho como app icon, favicon, avatar
  • 3 segmentos = 3 níveis do método
  • Lê-se como "C" mesmo em tamanho pequeno
  • Os gaps entre segmentos criam ritmo visual

Porquê este é o recomendado

  • Conta a história: 3 segmentos = Corrige, Constrói, Compete
  • Ícone standalone: funciona como app icon, pin, favicon
  • Thick lines: bold, confiante, funciona em qualquer tamanho
  • Escalável: do tamanho de moeda a billboard
  • Memorável: "aquele C partido em 3" — descritível em 3 segundos
  • Único: ninguém tem um C assim
Option C — Bold
The Badge
Emblema completo estilo vintage-moderno. O "C" domina o centro, "METHOD" em arco. Perfeito para merch, patches, t-shirts. Presença forte.

Notas Draplin

  • Badge style — funciona em patches, pins, t-shirts, stickers
  • Presença forte e autoritária
  • "CORRIGE · CONSTRÓI · COMPETE" integrado no logo
  • Double border = toque vintage-moderno
  • Dots laterais = separadores clássicos
  • Precisa de versão simplificada para tamanhos pequenos (usa só o C)
