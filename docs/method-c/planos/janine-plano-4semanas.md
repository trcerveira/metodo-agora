# Method C — Training Plan: Janine

**Start:** March 30, 2026 | **Duration:** 4 weeks | **Frequency:** 5x/week
**Level:** 1 — Foundation | **Type:** Full Body (1 workout, repeat every day) | **Duration:** 60 min

**Assessment:** Ankle poor, knee fair, thumb pain, low strength, poor endurance
**Goals:** Lower body fat, lower stress, build muscle, cardio condition

---

## THE WORKOUT (~60 min)

### Warm-Up (7 min)

| Exercise | Sets x Reps | Notes |
|----------|-------------|-------|
| Diaphragmatic breathing | 10 breaths | 4s inhale, 6s exhale |
| Cat-Cow | 2x8 | Slow and controlled |
| Ankle circles | 10 each foot | Full range, no pain |

### Foot & Knee Stability (12 min)

| Exercise | Sets x Reps | Notes |
|----------|-------------|-------|
| Short foot (arch activation) | 2x10 each foot | Barefoot. "Shorten" the foot without curling toes |
| Ankle mobility wall drill | 3x8 each foot | Heel on ground, knee pushes toward wall |
| Single leg balance | 3x20s each foot | Barefoot. Eyes fixed on a point |
| Wall sit | 3x20s | Back flat against wall, knees at 90 degrees |

### Full Body Strength — Tempo 2-2 (28 min)

All exercises: **2 seconds up, 2 seconds down.**

| Exercise | Sets x Reps | Notes |
|----------|-------------|-------|
| Goblet squat (5L bottle at chest) | 3x10 | Go as deep as possible without pain |
| Split squat (static) | 3x8 each leg | Hold wall if needed |
| Glute bridge | 3x12 | Squeeze glutes 2s at the top |
| Incline push-up (closed fists) | 3x8 | Fists to protect thumbs |
| Single arm row (bottle/backpack) | 3x10 each arm | Light grip, thumb relaxed |
| Dead bug | 3x6 each side | Lower back pressed into floor |
| Calf raises (bilateral) | 3x12 | Tempo 2-2. Ankle reinforcement |

### Cardio (8 min)

| Exercise | Format | Notes |
|----------|--------|-------|
| Brisk walking / light jog | 2 min fast + 1 min normal x3 | Should be able to talk but with effort |

### Cool-Down (5 min)

| Exercise | Duration |
|----------|----------|
| Calf stretch against wall | 30s each |
| Pigeon stretch | 45s each side |
| Diaphragmatic breathing lying down | 2 min |

## Progression

| | Weeks 1-2 | Weeks 3-4 |
|--|-----------|-----------|
| Strength reps | 8-10 | 10-12 |
| Wall sit | 20s | 30s |
| Single leg balance | 20s eyes open | 30s or 10s eyes closed |
| Push-ups | Table (high) | Chair (lower) |
| Cardio | 2 min fast + 1 min normal x3 | 2 min fast + 1 min normal x4 |
| Glute bridge | Both legs | Try single leg |
| Calf raises | Bilateral | Try unilateral |

## Thumb Pain Modifications

| Normal exercise | Modification for Janine |
|----------------|------------------------|
| Push-up (open hands) | Closed fists or push-up handles |
| Farmer's carry (normal grip) | Hold with palm/fingers, thumb relaxed |
| Row with weight | Light grip, use straps if needed |
| Plank (hands) | Forearm plank |
| Mountain climbers | Fist support or forearms |

**Rule:** If any exercise causes thumb pain — stop and modify. Never push through pain.
