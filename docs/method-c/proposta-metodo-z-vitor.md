# MÉTODO Z — Sistema de Transformação Física

---

## O Problema que Resolvemos

- 73% dos portugueses são sedentários
- 36,6% têm dor lombar
- As pessoas desistem do treino em 3 meses porque **não vêem resultados mensuráveis**
- Os PTs vendem sessões avulsas sem sistema — é trocar tempo por dinheiro

---

## O Método Z — O que É

Um sistema completo de transformação física com **testes de avaliação objectivos**, **3 níveis de progressão**, e **prova mensurável de resultados**.

Não é "mais um PT". É um método com:

```
AVALIAÇÃO → TREINO POR NÍVEL → RE-AVALIAÇÃO MENSAL → PROVA DE EVOLUÇÃO
```

O cliente **vê** a evolução com números. Não é "sinto-me melhor" — é "passei do nível 1 para o nível 2 com dados concretos."

### Categorias de Avaliação

- **Mobilidade e Estabilidade** — testes para avaliar a qualidade de movimento
- **Força** — capacidade muscular real
- **Resistência** — capacidade cardiovascular
- **Flexibilidade** — amplitude de movimento

### 3 Níveis de Progressão

Nível 1 → Nível 2 → Nível 3

Cada nível tem critérios de passagem objectivos. O cliente sabe exactamente onde está e para onde vai.

### Re-Avaliação Mensal

Todos os meses, o cliente repete os testes. Isto cria:

- **Prova objectiva** de que o método funciona
- **Motivação contínua** — o cliente vê a evolução mês a mês
- **Retenção** — quem vê resultados não cancela
