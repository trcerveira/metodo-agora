## 1. Visão

### O Problema

Solopreneurs que querem usar IA para construir negócios enfrentam 3 barreiras:

1. **Não sabem por onde começar** — a IA é poderosa mas esmagadora
2. **Não têm equipa** — trabalham sozinhos, fazem tudo
3. **Não sabem orquestrar** — mesmo com ferramentas, falta o sistema

O mercado está cheio de cursos que ensinam a "usar ChatGPT". Mas **ninguém vende a equipa pronta**.

### A Solução

O **Squad Agora OX** é um squad de 13 agentes AI especializados, com personalidades baseadas em figuras históricas portuguesas, prontos a instalar no terminal e a funcionar em 15 minutos.

> Não vendes um curso sobre como pescar. Vendes a cana, o anzol, e o isco. Tudo montado.

### Proposta de Valor

| Para quem | O que recebe | Resultado |
|-----------|-------------|-----------|
| Solopreneur sem equipa | 13 agentes especializados | Uma equipa virtual completa |
| Criador de conteúdo | Agentes de pesquisa, conteúdo, crescimento | Produção 10x mais rápida |
| Consultor/freelancer | Agentes de produto, design, qualidade | Entrega profissional sem equipa |

### Analogia Central

> A Levi Strauss não ensinou as pessoas a costurar calças. Vendeu as calças prontas a vestir. O Squad Agora OX não ensina a criar agentes. Entrega os agentes prontos a trabalhar.

## 5. Requisitos Técnicos

### 5.1 Formato do Produto

O Squad Agora OX é entregue como **um conjunto de ficheiros Markdown** que o Orquestrador instala no seu projecto Claude Code.

```
squad-agora-ox/
├── README.md                    # Guia de instalação (5 min)
├── config.yaml                  # Configuração do squad
├── agents/
│   ├── squad-chief.md           # D. Afonso Henriques
│   ├── analyst.md               # Fernando Pessoa ✅
│   ├── pm.md                    # Marquês de Pombal ✅
│   ├── po.md                    # D. Dinis
│   ├── sm.md                    # Aristides de Sousa Mendes
│   ├── architect.md             # Álvaro Siza Vieira
│   ├── dev.md                   # Bartolomeu de Gusmão
│   ├── data-engineer.md         # Pedro Nunes
│   ├── ux-designer.md           # Almada Negreiros
│   ├── qa.md                    # Eça de Queirós
│   ├── devops.md                # Infante D. Henrique
│   ├── content-creator.md       # Gil Vicente
│   └── growth.md                # Fernão de Magalhães
└── templates/                   # Templates de documentos
    ├── prd-tmpl.yaml
    ├── story-tmpl.yaml
    └── brief-tmpl.yaml
```

### 5.2 Instalação

O Orquestrador instala copiando a pasta para `.claude/commands/` no seu projecto:

```bash
# Passo 1: Descarregar
git clone https://github.com/trcerveira/squad-agora-ox.git

# Passo 2: Copiar para o projecto
cp -r squad-agora-ox/ .claude/commands/AgoraOX/

# Passo 3: Usar
# No Claude Code, activar com /AgoraOX:agents:nome-do-agente
```

### 5.3 Pré-Requisitos do Cliente

- Claude Code instalado e funcional
- Subscrição Claude (Pro ou Team)
- Terminal básico (Git Bash, Terminal, ou similar)
- Nenhum conhecimento de programação necessário

### 5.4 Actualizações (Subscrição Pro)

- Entrega via repositório privado no GitHub
- `git pull` para receber actualizações
- Changelog mensal com novas features e melhorias
- Novos agentes e templates adicionados periodicamente

## 6. Requisitos Funcionais

### RF01 — Cada agente deve activar-se com uma saudação padronizada
- Formato: `{icon} **{Nome} — {Papel} · Agora OX**` + `Qual é a tua missão?`
- Depois PÁRA e espera input do Orquestrador

### RF02 — Cada agente deve ter comandos com prefixo `*`
- Comandos em português de Portugal
- Lista acessível via `*help`
- Todos os agentes partilham: `*help`, `*sair`

### RF03 — Cada agente deve colaborar com os outros
- Tabela de colaboração em cada ficheiro
- Handoff claro: "Para isto, fala com {agente}"
- O Squad Chief coordena quando há dúvida

### RF04 — Linguagem sempre em Português de Portugal
- Zero inglês na interface com o Orquestrador
- Termos técnicos mantidos quando não há tradução natural
- Tom directo, sem jargão desnecessário

### RF05 — Nunca revelar sistemas internos
- Nunca mencionar AIOS, Synkra, ou nomes de frameworks internos
- O Orquestrador vê apenas "Agora OX"

### RF06 — Paleta e branding consistentes
- Verde `#BFD64B` · Preto `#0A0E1A` · Terminal `#111628` · Ivory `#F0ECE4`
- Todas as referências visuais usam esta paleta

## 7. Requisitos Não-Funcionais

### RNF01 — Instalação em menos de 15 minutos
- README claro com passos numerados
- Vídeo de instalação opcional

### RNF02 — Funcionar offline
- Ficheiros Markdown locais, sem dependência de servidor
- Só precisa de conexão para Claude Code (API da Anthropic)

### RNF03 — Compatível com qualquer projecto
- Agnóstico de linguagem e framework
