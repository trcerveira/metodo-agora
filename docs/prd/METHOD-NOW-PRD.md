## 1. Visão

### O Problema

Estamos no início da curva de adopção de IA. Toda a gente sabe que precisa. Quase ninguém sabe usar.

1. **ChatGPT criou a ilusão** — pessoas pensam que IA = fazer perguntas a um chatbot
2. **Ninguém ensina a CONSTRUIR** — cursos ensinam a "usar prompts", não a montar sistemas
3. **Claude Code é invisível** — a ferramenta mais poderosa do mercado e quase ninguém sabe que existe
4. **Solopreneurs trabalham sozinhos** — fazem tudo: vendas, produto, marketing, ops, finanças
5. **Contratar é caro e arriscado** — um funcionário custa $2K+/mês. Um agent custa $0.

### A Oportunidade

Na corrida ao ouro, vende pás.

- 62% dos portugueses já usam IA generativa (maior taxa da Europa)
- Apenas 11.5% das empresas implementaram IA nos processos
- 69% das PMEs perdem 5h/dia em tarefas repetitivas
- O mercado global de AI agents vai de $5B (2025) para $47B (2030)
- Claude Code + OpenClaw permitem deployar agents autónomos 24/7 — quase ninguém sabe fazer isto

### A Solução

**Method NOW** é um sistema que transforma solopreneurs em orquestradores de 13 agents de IA. Não é um curso. É uma equipa.

O cliente recebe:
- Um coach IA conversacional (NOW bot) que o guia passo a passo
- 13 agents especializados que trabalham para ele no Claude Code
- Um assessment de Zona de Genialidade que lhe diz ONDE focar
- Tudo via Telegram. Zero apps novas. Zero código.

### Proposta de Valor

> "You won't be replaced by AI. You'll be replaced by someone who uses AI. Method NOW makes you that someone."

| Para quem | O que recebe | Resultado |
|-----------|-------------|-----------|
| Solopreneur sem equipa | 13 AI agents + NOW bot coach | Uma equipa virtual completa |
| Freelancer que quer escalar | Sistema de orquestração + assessment | Escala sem contratar |
| Side-hustler | Step-by-step guiado por IA | Negócio montado em semanas |

### Analogia Central

> A Levi Strauss não escavou ouro. Vendeu as calças aos mineiros. O Method NOW não vende IA. Vende o sistema para USAR IA. A pá que ensina a escavar.

## 4. Teoria Base — MaaS (Method as a Service)

### O Problema da Transferência de Conhecimento

Cursos tradicionais transferem ~19% do conhecimento. O aluno paga $2.000 e retém $380 de valor.

### A Equação MaaS

```
Ke = ((T × A(t) × M × AmpIA) × Lcg) / (Lci + Lce) × (1 ± σ)
```

| Variável | O que é | Curso normal | Method NOW |
|----------|---------|-------------|------------|
| T | Qualidade do ensino | Video gravado | Frameworks codificados em agents |
| A(t) | Capacidade do aluno | Ignora — todos recebem o mesmo | NOW adapta ao nível e momento |
| M | Meio de comunicação | Video passivo (cano fino) | Chat interactivo (cano largo) |
| AmpIA | Amplificador IA | Zero | NOW traduz para a linguagem do aluno |
| Lcg | Prática real | "Agora faz o exercício" | NOW guia passo a passo, valida output |
| Lci+Lce | Ruído | 47 módulos, interface confusa | Uma conversa. Um passo. Zero ruído. |
| σ | Variação | Sem adaptação | NOW detecta se o aluno parou e ajusta |

**Resultado:** De 19% para 60-75% de conhecimento efectivamente transferido.

### Aplicação no Produto

- NOW bot É o amplificador (AmpIA)
- Chat conversacional maximiza M (meio interactivo)
- Passo a passo reduz ruído (Lci+Lce)
- Prática guiada maximiza Lcg
- Adaptação ao aluno controla σ
